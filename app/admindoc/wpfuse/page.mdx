# WPF Használati útmutasító

## Admin Oldal

1. **Az alkalmazás elindítása után egy bejelentkezési felület fog megjelenni amibe be kell jelentkezni egy admin vagy moderator szerepű felhasználóval (Az adminpage eléréséhez adminnal kell bejelentkezni)**

2. **Teszt felhasználók:**
   - TestAdmin - Jelszó123@ (Admin felhasználó)
   - TestModerator - Jelszó123@ (Moderátor felhasználó)
   
### Adminos bejelentkezés után az admin page fog elénk kerülni

1. **A Felhasználók Betöltése gombra nyomva betöltjük az összes létező felhasználót**
2. **A Felhasználó Elnémítása gombra nyomva elnémítjuk a felhasználót, ezek után nem fog tudni kommentelni**
3. **A Felhasználó Kitiltása gombra nyomva kitiltjuk a felhasználót, ezek után nem fog tudni bejelentkezni**
4. **Szerepkör változtatása**
    - Elsőnek ki kell választani a felhasználót akinek meg kívánjuk változtatni a szerepkörét
    - Aztán a legördülő listában a Szerepkör Változtatása gomb alatt kiválasztjuk melyik szerepkörre akarjuk megváltoztatni
    - Utolsó sorban pedig megnyomjuk a Szerepkör Változtatása gombot, hogy végrehajtsuk a műveletet
5. **A Kérelmek gombra nyomással átléphetünk a Kérelmek Oldalra, a következő szegmens erről az oldaról fog szólni**
6. **A Kilépés gombbal visszatérhetünk a bejelentkezés oldalra**

## Kérelmek Oldal

1. **A Függőben Lévő Kérelmek Betöltése gombra nyomva betöltjük az összes létező kérelmet**
2. **A Kérelem Elfogadása gombra nyomva elfogadjuk a kérelmet és a felhasználó szerepköre megváltozik arra amit kért( 1 = Author, 2 = Critic)**
3. **A Kérelem Elutasítása gombra nyomva elutasítjuk a kérelmet és nem változik meg a felhasználó szerepköre**
4. **A Vissza gombbal visszatérhetünk az admin oldalra**

## Moderátor Oldal

1. **A Függőben Lévő Jelentések Betöltése gombra nyomva betöltjük az összes létező bejelentést**
2. **A Bejelentés Elfogadása(Némítás) gombra nyomva elfogadjuk a bejelentést ezzel elnémítva ezt a felhasználót**
3. **A Bejelentés Elutasítása gombra nyomva elutasítjuk a bejelentés**
4. **A Kilépés gombbal visszatérhetünk a bejelentkezés oldalra**
