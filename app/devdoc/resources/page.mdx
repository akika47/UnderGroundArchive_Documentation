    ## Függőségek

    ### 1. @react-oauth/google (0.12.1)
    **Google bejelentkezés integráció React alkalmazásokba**  
    Egy React komponens, ami egyszerűvé teszi a Google OAuth 2.0 alapú bejelentkezés implementálását. Lehetővé teszi:
    - Google-fiókkal való bejelentkezést
    - Felhasználói profil adatok lekérését
    - Token kezelést
    - Egyszerű integráció React alkalmazásokba

    ### 2. framer-motion (12.0.5)
    **Animációs könyvtár React alkalmazásokhoz**  
    Egy fejlett animációs könyvtár, ami segít:
    - Sima, természetes animációk létrehozásában
    - Komponensek animálásában egyszerű deklaratív szintaxissal
    - Áttűnések, mozgások, layout animációk készítésében
    - Komplex interakciók és mikrointerakciók megvalósításában

    ### 3. jsonwebtoken (9.0.2)
    **JSON Web Token (JWT) kezelő könyvtár**  
    JWT tokenek generálására és ellenőrzésére szolgáló eszköz:
    - Tokenek aláírása (signing)
    - Tokenek érvényesítése (verification)
    - Tokenek dekódolása
    - Titkosítási algoritmusok támogatása (HS256, RS256 stb.)
    - Lejárati idő kezelése

    ### 4. jwt-decode (4.0.0)
    **JWT tokenek dekódolása kliens oldalon**  
    Egy egyszerű eszköz JWT tokenek tartalmának kiolvasásához:
    - Fejléc (header) és adattartalom (payload) kibontása
    - Base64 dekódolás
    - Bármilyen környezetben használható (böngésző, Node.js)
    - Nincs függősége más könyvtárakra

    ### 5. simplebar (6.3.0)
    **Egyéni görgetősáv megoldás**  
    Natív scrollbar helyettesítése testreszabható megoldással:
    - Egységes megjelenés minden böngészőben
    - Teljes méretű tartalom görgetése fix magasságú elemekben
    - Testreszabható kinézet (CSS segítségével)
    - Könnyű súlyú megoldás

    ### 6. simplebar-react (3.3.0)
    **Simplebar React komponens változata**  
    A Simplebar React komponens formájában történő használatát teszi lehetővé:
    - React komponensként való használat
    - Egyszerű integráció React alkalmazásokba
    - Minden Simplebar funkció elérhető React környezetben
    - Gyermek komponensek automatikus kezelése
