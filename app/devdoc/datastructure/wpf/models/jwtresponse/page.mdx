# JwtResponse Model Dokumentáció

## **Áttekintés**
A `JwtResponse` osztály egy egyszerű adatmodell, amelyet a szerverről érkező JWT (JSON Web Token) válaszok deszerializálására használnak a WPF alkalmazásban. Ez az osztály a hitelesítési folyamat során kapott válaszok kezelését segíti.

---

## **Tulajdonságok**

| Tulajdonság | Típus   | Leírás                                                                                         |
|-------------|---------|------------------------------------------------------------------------------------------------|
| `result`    | string  | A szerver által visszaadott JWT token vagy hitelesítési válasz szövege. Fontos szerepe van a felhasználói munkamenet kezelésében. |
